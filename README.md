CyberVault is a lightweight, user-friendly desktop application that allows users to verify the integrity of files using cryptographic hash functions. It also includes a unique real-time file monitoring feature that detects unauthorized modifications and alerts the user.

🌟 Features:
🔐 Generate secure hash values (SHA-512, SHA-256, SHA-1, MD5) for any file.

🕵️‍♂️ Real-time monitoring of files for unauthorized changes.

📋 Copy hash values to the clipboard with one click.

🖥️ Simple, intuitive graphical interface built with Tkinter.

🛡️ Instant alerts via popup and status bar when a file is modified.

📂 Supports files of any size, with chunked reading for performance.
